# AI-Powered Annual Appeal Optimizer

Interactive prototype demonstrating AI-assisted donor segmentation and personalized appeal generation for Catholic dioceses, parishes, and faith-based organizations.

## Overview

This tool helps Catholic fundraising professionals optimize their annual appeal campaigns by:

- Analyzing parishioner and donor segments based on giving history and engagement
- Generating personalized messaging strategies aligned with parish communication styles
- Recommending optimal ask amounts and outreach channels
- Providing timing suggestions based on liturgical calendar and donor behavior

## Live Demo

[Launch the AI Appeal Optimizer](https://jesimpsonjr-prodmgmt.github.io/ai-appeal-optimizer/)

## Technical Details

- Self-contained HTML application (no build process required)
- Integrates with Anthropic Claude API for AI-powered recommendations
- Client-side processing for data privacy

## Context

Built as a product concept prototype for the Catholic church management and fundraising software market (Pushpay/ParishStaq ecosystem). Demonstrates practical application of AI to solve real parish fundraising challenges.
